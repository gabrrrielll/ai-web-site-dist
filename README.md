# AI Website Builder - Public Build Files

This repository contains the public build files for the AI Website Builder application.

## 🚀 Deployment

This repository is automatically updated with the latest build from the private main repository.

### For cPanel Deployment

Use this repository URL in cPanel Git Version Control:

```
https://github.com/gabrrrielll/ai-web-site-dist.git
```

### Direct File Access

You can clone this repository directly to your server:

```bash
git clone https://github.com/gabrrrielll/ai-web-site-dist.git
```

## 📁 Contents

- `index.html` - Main application file
- `assets/` - Compiled CSS, JS, and other assets
- `manifest.json` - PWA manifest
- `sw.js` - Service worker for offline functionality
- `.htaccess` - Apache configuration
- `nginx.conf` - Nginx configuration
- `robots.txt` - SEO robots file
- `sitemap.xml` - SEO sitemap

## 🔄 Auto-Update

This repository is automatically updated when:
1. New builds are generated from the main application
2. Changes are pushed to the private repository
3. The deploy script runs successfully

## 📝 Notes

- This repository contains only the compiled/built files
- Source code is maintained in a separate private repository
- All sensitive information has been removed from these files
